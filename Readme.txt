# 💼 Raj Sojitra - Developer Portfolio

Welcome to the official repository for my personal portfolio website, built to showcase my skills, projects, and services as a web developer.

🔗 **Live Site**: https://raj-sojitra-portfolio.vercel.app/

## 📌 Features

- Responsive and modern design
- Interactive 3D skill section (HTML, CSS, JS, PHP, Laravel)
- Projects with image sliders and GitHub/source code links
- Services and client testimonials
- Contact form with working submission
- Smooth navigation and section-based layout
- Built using HTML, CSS, JavaScript, and Bootstrap


## 🧑‍💻 Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- AOS (Animate on Scroll)
- Boxicons / Bootstrap Icons
